# 🦠 Disease-prediction-ml-typhoid


This project applies Machine Learning techniques to predict Typhoid Fever status in adolescents using clinical and symptom-based data.

## 📊 Project Overview

Typhoid fever remains a major public health concern in developing countries. Early detection is often difficult due to limited diagnostic resources. This project builds a predictive model to assist in early classification of typhoid cases using machine learning.

## 🎯 Objectives

- Build a predictive model for typhoid diagnosis
- Compare multiple ML algorithms:
  - Logistic Regression
  - Random Forest
  - Support Vector Machine (SVM)
- Evaluate model performance using standard metrics

## 🧠 Machine Learning Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Selection
4. Model Training
5. Model Evaluation
6. Model Comparison

## 🧪 Models Used

- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

## 📈 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
